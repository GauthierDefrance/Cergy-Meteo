Detailed Description

Gestion des données géographiques et météorologiques.

Ce fichier définit des fonctions permettent de récuperer des informations sur les régions, départements et villes de France dans des fichiers CSV, et de les retourner sous différentes formes pour un accès facile. Il contient aussi des fonctions appellant l'API renvoyant des données météorologiques.
